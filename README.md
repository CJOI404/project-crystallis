# project-crystallis

This is a complete remake of the Final Fantasy 13 combat system for the Playstation Portable (PSP).

This can be played on real PSP hardware as well as PC, iOS, and Android through the PPSSPP emulator using the PSP.EBOOT file. 

FF13 has amazing fast-paced gameplay, but it is kind of ruined by how slow and linear the rest of the game is. Since this battle system died with FF13-2 I thought it would be cool to bring it back. The controls are also quite simple which makes it perfect for a PSP version in my eyes. 

Right now focus is solely on the gameplay. Once the combat loop is complete, I will improve the UI and maybe add 3d rendering instead of 2d sprites so that it mimics the actual original FF13. Team loadouts and enemies will be fully customizable and I will likely add levels or some kind of roguelite implementation towards the end of development. For now though the game will remain very ugly and basic as I get the combat fully functional, as the FF13 combat system is pretty in-depth. I might also stick to a 2D only if 3D turns out to be too big of a project (2D would not really impact the gameplay at all, though its less flashy).
